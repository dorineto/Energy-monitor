Para que os programas nesse projeto funcione corretamente é necessário:
 
  *Ter o SQLserver intalado em um computador, e executar o arquivo de Energy_monitor.sql no qual contém a estrutura do banco de dados do projeto. Caso precise de valores de exemplo execute também o arquivo de valores_exeplo.sql que insere valores de exemplo no banco de dados 

  *Ter o java 8 ou 13 instalado no computador que vai rodar os programas de Coletor, Painel ou Dashboard, também ter os driver de acessso do java ao SQLserver (JDBC) na verção de acordo com o java instado no computador.

  *Configurar coretamente o arquivo de configuração de cada aplicação.

Para excutar as aplicações de Coletor, Painel ou Dashboard utilize o arquivo .bat (no caso do windowns) ou o .sh (no caso de Linux) com os mesmos nomes das aplicações.

Caso o interesse de saber sobre as estruturas das aplicações tem os arquivos de codigo fonte nas pastas src e os modelos de classe ou lógico no diretério modelos.
